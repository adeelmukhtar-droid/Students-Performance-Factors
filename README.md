# 🎓 Student Performance Prediction using Machine Learning

## 📌 Project Overview

This project predicts students' exam performance using Machine Learning techniques. The goal is to analyze different student-related factors and build predictive models that estimate the final exam score.

The project was developed using Python in Google Colab with the Student Performance Factors dataset from Kaggle.

## 📊 Dataset

**Dataset:** Student Performance Factors

The dataset contains various academic, personal, and lifestyle factors that influence students' exam scores.

Target Variable:

- Exam_Score

Example Features:

- Hours Studied
- Attendance
- Previous Scores
- Sleep Hours
- Motivation Level
- Internet Access
- Teacher Quality
- Family Income
- School Type
- and more...

## 📈 Model Performance

| Model | R² Score |
|--------|----------:|
| Linear Regression | **0.6886** |
| Random Forest Regressor | **0.6533** |

Linear Regression achieved the highest R² score and was selected as the best-performing model for this dataset.
